# FileFlux

**Flow Between Formats**

FileFlux is a modern, responsive, client-side web application for file conversion and image compression. The app performs all processing locally in the browser, ensuring your files are never uploaded to a server!

🔗 **[Live Demo](https://anjanavp231.github.io/FileFlux/)**

## 🚀 Features

*   **Convert Image Formats:** Easily convert between JPG and PNG.
*   **Image to PDF:** Convert any JPG or PNG image into a high-quality PDF document.
*   **PDF to Image:** Extract the first page of a PDF document and convert it to a JPG or PNG.
*   **Image Compression:** Adjust the output quality (Low, Medium, High) to significantly reduce file sizes directly in your browser.
*   **Image Resizing:** Optionally resize image dimensions while converting.
*   **Completely Private:** 100% client-side processing. No data is sent to external servers.

## 🛠️ Tech Stack

*   **HTML5 & CSS3** for structure and modern UI/UX design (Glassmorphism layout).
*   **Vanilla JavaScript** for core logic handling without any frameworks.
*   [**Canvas API**](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API) for image processing and compression.
*   [**jsPDF**](https://github.com/parallax/jsPDF) for generating PDF documents from images.
*   [**PDF.js**](https://mozilla.github.io/pdf.js/) for rendering and extracting images from PDF files.

## 💻 Local Execution

Since everything runs in the browser, getting started is extremely simple:

1. Clone the repository:
   ```bash
   git clone https://github.com/AnjanaVP231/FileFlux.git
   ```
2. Navigate into the directory:
   ```bash
   cd FileFlux
   ```
3. Open `index.html` in your favorite web browser!
